Load file screen



  • Back tab: back to previous main screen.
  • Text field: show a file name.
  • Load button: load a sound file to corresponding pad number.
  • Arrows: change pad number.
  • Directory or file names: tap a direcotry to open a sub directory or file to select.
  • Swipe horizontally to go to other pages.
