Q_open

constructs DOM tree.

Example


Q_open("http://example.com")
## OR
parser = Parser()
parser.set_encoding('utf-8')
Q_open("http://example.com", parser)

Arguments

  • Name: url, Type: str
    • url
  • Name: parser, Type: Parser
    • parser

Source


def Q_open(url, parser = None):
    global _parser
    if parser is not None:
        if _parser is not None:
            _parser.close()
            _parser.clear()
        _parser = parser
        _parser.open(url)
    else:
        if _parser is None:
            _parser = Parser()
            _parser.open(url)
        else:
            _parser.close()
            _parser.clear()
            _parser.open(url)
    _parser.construct()
