attr

returns attribute value or space separated values

Example


parser = Parser()
parser.open("http://example.com")
link = parser.tag('a').first().attr('href')
parser.close()

Arguments

  • Name: key, Type: str
    • attribute name

Source


def attr(self, key):
    if not key in self._attr: return None
    else: return self._attr[key]
