attr

returns a attribute value or space separated values.

Example


Q_open("http://example.com")
link = Q('a:first').attr('href')
Q_close()

Arguments

  • Name: key, Type: str
    • attribute name

Source


def attr(self, key):
    if not key in self._attr: return None
    else: return self._attr[key]
