set_attr

[internal] set an attribute.

Arguments

  • Name: key, Type: str
    • attribute name
  • Name: value, Type: str
    • attribute value

Source


def set_attr(self, key, value):
    if not key in self._attr: self._attr[key] = value
    else: self._attr[key] += ' ' + value
