select_prev_sibling

returns the sibling that is from previous element.

Example


Q_open("http://example.com")
fourth = Q('ul:first > li:nth-child(4)')
li_0_2 = fourth.select_prev_sibling()
Q_close()

Source


def select_prev_sibling(self):
    ret = pyNodeList()
    n = self.prev_all()
    if isinstance(n, pyNode):
        ret.append(n)
    else:
        for elem in n:
            ret.append(elem)
    if len(ret) == 1: return ret[0]
    else: return ret
