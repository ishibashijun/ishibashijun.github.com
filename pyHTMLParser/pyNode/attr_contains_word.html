attr_contains_word

returns True if the attribute value contains the given word.

Example


Q_open("http://example.com")
logos = Q('img').attr_contains_word('alt', 'logo')
Q_close()

def attr_contains_word(self, attr, value):
    a = self.attr(attr)
    if a is not None:
