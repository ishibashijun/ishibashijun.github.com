has_cls

returns True if the element has a specified class attribute value.

Example


Q_open("http://example.com")
link_class = Q('a:first').has_cls("link")
Q_close()

Arguments

  • Name: cls, Type: str

Source


def has_cls(self, cls):
    if 'class' in self._attr:
        cls_name = re.compile(cls)
        if cls_name.search(self._attr['class']) is not None:
            return True
    return False
