__ne__

returns true if it does not equal

Example


parser = Parser()
parser.open("http://example.com")
div_first = parser.tag('div').first()
id_main = parser.id('main')
print(str(div_first != id_main))
parser.close()

>> True

Source


def __ne__(self, node):
    return not self.__eq__(node)
