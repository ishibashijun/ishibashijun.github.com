has_class

returns True if element has specified class attribute

Example


parser = Parser()
parser.open("http://example.com")
link_class_names = parser.tag('a').first().cls()

Arguments

  • Name: cls, Type: str

Source


def has_class(self, cls):
    if 'class' in self._attr:
        return self._attr['class'].find(cls) != -1
    return False
