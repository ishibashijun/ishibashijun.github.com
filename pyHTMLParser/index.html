pyHTMLParser

Python 3.x library for html parsing

Aboud pyHTMLParser

A simple html parser that constructs DOM tree. It aims to provide jquery like API thus it has features like chaining methods.

The library is still in early stage of development. If you found bugs or new features that wanted to be added, you can report on issues on GitHub.
This documentation is still in development. If you also found inappropriate expression, you can report on issues.

Installation


>> pip install pyHTMLParser
