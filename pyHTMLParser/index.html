pyHTMLParser

Python 3.x library for html parsing

Aboud pyHTMLParser

A simple html parser that constructs DOM tree. It is aimed to provide jquery like API.

The library is still in early stage of development. If you found bugs or new features that wanted to be added, Please report on issues on GitHub.
This documentation is still in development. If you also found inappropriate expression, Please report on issues.

Installation


>> pip install pyHTMLParser
