handle_data

[internal] handles arbitrary data when constructing DOM tree.

Arguments

  • Name: data, Type: str
    • content of the arbitrary data

Source


def handle_data(self, data):
    if self._is_started:
        text = data.replace('\r\n', '')
        text = text.replace('\n', '')
        text = text.replace('\t', '')
        self._dom[-1].add_text(text.strip())
