cls

returns elements matching a class name

Example


parser = Parser()
parser.open("http://example.com")
tabs = parser.cls('tab')
parser.close()

Arguments

  • Name: c, Type: str
    • class name

Source


def cls(self, c):
    return self._nodes.cls(c)
