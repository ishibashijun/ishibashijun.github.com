tag

[internal] returns elements that are matching the tag name.

Arguments

  • Name: tag, Type: str
    • tag name

Source


def tag(self, tag):
    ret = pyNodeList()
    for node in self._nodes:
        if node.name() == tag.lower():
            ret.append(node)
    return ret
