handle_startendtag

[internal] handles xhtml like self closing tags when constructing DOM tree

Arguments

  • Name: tag, Type: str
    • tag name
  • Name: attrs, Type: 2D list
    • [0] attribute name, [1] attribute value

Source


def handle_startendtag(self, tag, attrs):
    if self._is_started:
        node = pyNode(tag.lower())
        node.set_parent(self._dom[-1])
        self._nodes.append(node)
        for attr in attrs:
            node.set_attr(attr[0], attr[1])
