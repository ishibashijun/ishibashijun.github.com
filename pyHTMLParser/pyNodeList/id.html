id

returns a element matching a id name

Example


parser = Parser()
parser.open("http://example.com")
divs = parser.tag('div')
nav = divs.id('nav')
parser.close()

Arguments

  • Name: i, Type: str
    • id name

Source


def id(self, i):
    for node in self:
        if node.get_attr('id') == i:
            return node
    return None
