id

returns an element that is matching the id name.

Example


Q_open("http://example.com")
div_main = Q('div').id('main')
Q_close()

Arguments

  • Name: i, Type: str
    • id name

Source


def id(self, i):
    for node in self:
        if node.get_attr('id') == i:
            return node
    return None
