descendant_class

returns descendant elements matching a class name

Example


parser = Parser()
parser.open("http://example.com")
ul = parser.tag('ul')
links = ul.descendant_class('link')
parser.close()

Arguments

  • Name: cls, Type: str
    • class name

Source


def descendant_class(self, cls):
    ret = pyNodeList()
    for node in self:
        descendant = node.descendant_class(cls)
        if len(descendant) != 0:
            ret.extend(descendant)
    return ret
