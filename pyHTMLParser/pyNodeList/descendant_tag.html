descendant_tag

returns descendant elements matching a tag name

Example


parser = Parser()
parser.open("http://example.com")
ul = parser.tag('ul')
a = ul.descendant_tag('a')
parser.close()

Arguments

  • Name: tag, Type: str
    • tag name

Source


def descendant_tag(self, tag):
    t = tag.lower()
    ret = pyNodeList()
    for node in self:
        descendant = node.descendant_tag(t)
        if len(descendant) != 0:
            ret.extend(descendant)
    return ret
