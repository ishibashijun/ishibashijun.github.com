:nth-last-of-type(n)

returns elements that are the specified index of type from last.

Example


Q_open("http://example.com")
second_from_last = Q('article:nth-last-of-type(2)')
fifth_from_last = Q('div:nth-last-of-type(5)')
Q_close()
