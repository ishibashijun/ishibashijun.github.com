not_tag

returns elements not matching a tag name

Example


parser = Parser()
parser.open("http://example.com")
div = parser.tag('div')
not_links = div.children().not_tag('a')
parser.close()

Arguments

  • Name: tag, Type: str
    • tag name

Source


def not_tag(self, tag):
    t = tag.lower()
    ret = pyNodeList()
    for node in self:
        if node.name() != t:
            ret.append(node)
    return ret
