not_eq

returns elements not a specified index element

Example


parser = Parser()
parser.open("http://example.com")
ul = parser.tag('ul')
li_not_three = ul.children().not_eq(3)
parser.close()

Arguments

  • Name: i, Type: int
    • index

Source


def not_eq(self, i):
    ret = pyNodeList()
    ret.extend(self.lt(i))
    ret.extend(self.gt(i))
    return ret
