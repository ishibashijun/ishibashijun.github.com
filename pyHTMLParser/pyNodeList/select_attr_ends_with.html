select_attr_ends_with

returns elements that have the specified attribute and the value that ends with the given value.

Example


Q_open("http://example.com")
logo = Q('img').select_attr_ends_with('alt', 'logo')
Q_close()

Arguments

  • Name: attr, Type: str
    • attribute
  • Name: value, Type: str
    • value

Source


def select_attr_ends_with(self, attr, value):
    ret = pyNodeList()
    for node in self:
        if node.attr_ends_with(attr, value):
            ret.append(node)
    if len(ret) == 1: return ret[0]
    else: return ret
