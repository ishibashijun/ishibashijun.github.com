eq

returns a single element at the specified index

Example


parser = Parser()
parser.open("http://example.com")
li = parser.tag('ul').id('list').child('li')
third_element_of_list = li.eq(3)
parser.close()

Arguments

  • Name: index, Type: int
    • index of the pyNodeList

Source


def eq(self, index):
    if len(self) > abs(index):
        return self[index]
    return None
