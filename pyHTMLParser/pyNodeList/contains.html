contains

returns elements that are containing the specified text.

Example


Q_open("http://example.com")
greeting = Q('p').contains('Hello World')
Q_close()

Arguments

  • Name: text, Type: str
    • text to search to

Source


def contains(self, text):
    ret = pyNodeList()
    for node in self:
        if node.text().find(text) != -1:
            ret.append(node)
    if len(ret) == 1: return ret[0]
    else: return ret
