select_attr_contains_word

returns elements that have the specified attribute and the value that contains the given word.

Example


Q_open("http://example.com")
logo = Q('img').select_attr_contains_word('alt', 'logo')
Q_close()

Arguments

  • Name: attr, Type: str
    • attribute
  • Name: value, Type: str
    • value

Source


def select_attr_contains_word(self, attr, value):
    ret = pyNodeList()
    for node in self:
        if node.attr_contains_word(attr, value):
            ret.append(node)
    if len(ret) == 1: return ret[0]
    else: return ret
