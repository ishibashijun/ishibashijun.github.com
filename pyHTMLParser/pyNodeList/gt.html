gt

returns elements greater than specified index

Example


parser = Parser()
parser.open("http://example.com")
li = parser.tag('ul').id('list').child('li')
elems = li.gt(3)
parser.close()

Arguments

  • Name: index, Type: int
    • index of the pyNodeList

Source


def gt(self, index):
    return self[index:]
