lt

returns elements that are the less than specified index.

Arguments

  • Name: index, Type: int
    • index of the pyNodeList

Source


def lt(self, index):
    li = self[:index]
        ret = pyNodeList()
        for node in li:
            ret.append(node)
        return ret
