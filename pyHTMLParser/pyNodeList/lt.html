lt

returns elements less than specified index

Example


parser = Parser()
parser.open("http://example.com")
li = Parser.tag('ul').id('list').child('li')
elems = li.lt(3)
parser.close()

Arguments

  • Name: index, Type: int
    • index of the pyNodeList

Source


def lt(self, index):
    return self[:index]
