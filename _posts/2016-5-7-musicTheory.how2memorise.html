Studied a little of music theory. This is how I memorise.

Mode

Memorise initilas of mode => IDPLMAL

  • ID

    - Id or index because those are used everywhere.

  • PL

    - Imagine if there was a openPL open source library.

  • MAL

    - Imagine if there was a openMAL open source library.

Modes

  • Ionian

    - Ion a molecule from chemistry and all the mode's suffix is same.

  • Dorian

    - Pro Bodybuilder Dorian Yates.

  • Phrygian

    - Pyramids in Egypt.

  • Lydian

    - Lydia, a feminine first name.

  • Mixolydian

    - It is easy to associate with mixing in music.

  • Aeolian

    - Only vowels.

  • Locrian

    - Locrian rocks because locrian's "rian" starts with "r".


Circle of fifths

Clockwise

  • CG

    - Computer graphics.

  • DAE

    - It's ".dae", file extension for COLLADA file format, Khronos Group developing.

  • BF

    - Reading file needs buffer.

Counter-clockwise

  • CF

    - Confer, "cf.".

  • BEAD G

    - Protein G Magnetic Beads.

Minor

  • Rotate major to -90 degree.
  • When "BEAD G" "E" have reached at bottom spread "A" to both side then meets at where "E" is with "D".
Every direction ends with "C".
